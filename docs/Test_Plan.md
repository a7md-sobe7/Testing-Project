# Test Plan - MedAi Application

## 1. Introduction
This document outlines the testing strategy for the MedAi Medical Appointment System.

## 2. Scope
### In Scope
*   **Registration Form**: Validation of personal information inputs.
*   **Contact Form**: Verification of inquiry submission.
*   **UI Elements**: Checking visibility of key fields.

## 3. Test Strategy
*   **Automated Testing**: Cypress scripts executing against the local server.
*   **Environment**: `http://127.0.0.1:5501`

## 4. Tools
*   **Cypress**: For E2E automation.
*   **Mochawesome**: For reporting.

## 5. Deliverables
*   Test Scripts (`medai_*.cy.js`)
*   Execution Report (HTML)
